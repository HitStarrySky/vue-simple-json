<p align="center">vue-simple-json</p>

> 极简的 jsonEditor 插件

## 安装

```shell

# pnpm

pnpm add vue-simple-json

```

## 特性

- ✊ Vue 3 Composition API
- 🔥 TypeScript 开发，提供完整的类型定义文件。

## 功能支持

- 节点自定义
- 自定义输入类型 可配合主流 UI 框架
- 自定义 key 值颜色
- 展开层级
- 自定义操作
